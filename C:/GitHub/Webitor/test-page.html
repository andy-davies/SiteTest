About This Test Page

This is a test page for the Webitor browser extension. You can click on any element with the data-editable attribute to edit its content directly in the browser.

The page also uses CSS variables (see :root in the styles) which you can modify in real-time using the CSS Variables editor in the side panel. Try changing colors, sizes, or spacing!

Editable Demo Section

This entire section is editable. You can modify the heading above and this paragraph. The changes will be reflected immediately on the page.

When you're done editing, you can commit your changes to GitHub using the commit form in the side panel.

CSS Variables to Try

This page uses the following CSS variables that you can modify:

  • --primary-color: The primary blue color (currently blue)
  • --secondary-color: The secondary green color
  • --background-color: The page background color
  • --text-color: The main text color
  • --heading-size: The size of main headings
  • --body-size: The size of body text
  • --spacing: The spacing between elements

Features to Test

  1. Editing Mode: Toggle editing mode on/off
  2. Click to Edit: Click any element with data-editable attribute
  3. CSS Variables: Modify CSS variables and see real-time changes
  4. HTML Snapshot: Capture the edited page as HTML
  5. GitHub Commit: Commit changes to a GitHub repository
  6. Offline Mode: Save changes offline and sync later

Note: Changes made on this page are not persistent. If you refresh the page, your edits will be lost unless you commit them to GitHub first.
