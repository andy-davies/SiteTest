# Webitor Primary School Website

A modern, multi-page website for Webitor Primary School, designed with clean, professional styling suitable for a UK primary school.

## Features

- **7 Complete Pages:**
  - Home (index.html) - Welcome page with hero section, features, and news preview
  - About Us (about.html) - School vision, values, history, and leadership team
  - Curriculum (curriculum.html) - Core subjects, enrichment activities, and EYFS information
  - Admissions (admissions.html) - Application process, criteria, and school tours
  - Parents Hub (parents.html) - Term dates, routines, wraparound care, and policies
  - News & Events (news.html) - Latest news articles and upcoming events calendar
  - Contact (contact.html) - Contact information, form, and key staff contacts

- **Modern CSS Design:**
  - Fully responsive layout (mobile, tablet, desktop)
  - Professional color scheme (navy blue, light blue, golden yellow accents)
  - Modern card-based layouts
  - Smooth transitions and hover effects
  - Grid-based responsive design
  - Accessible and clean typography

- **Key Sections:**
  - Sticky navigation header
  - Hero sections with gradients
  - Feature cards
  - News/events grids
  - Staff profiles
  - Contact forms
  - Footer with quick links

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate between pages using the header navigation
3. All pages are fully linked and functional

## Images

Images sourced from Ark Conway Primary Academy website (https://arkconway.org/) as demonstration placeholders. In production, these should be replaced with actual school photos.
